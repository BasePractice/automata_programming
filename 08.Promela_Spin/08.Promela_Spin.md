


### Получения графа из модели

Сначала устанавливается пакет `graphviz`
```bash
pacman -S mingw-w64-x86_64-graphviz
```

```bash
./pan -D > pan.dot
dot -Tps pan.dot -o pan.ps
dot -Tpng pan.dot > pan.png
```
